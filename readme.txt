==Description==

Ruven Toolkit provides the following extra functionality to the collection of Ruven Themes:

* Portfolio and Gallery Post Types
* Custom Widgets
* Shortcode Editor
* Post Format UI

==Frequently Asked Questions==

= What is this plugin and why do I need it? =

Ruven Toolkit provides extra functionality to the collection of Ruven Themes. The toolkit adds a Portfolio and Gallery post type, custom widgets, a post format UI, and shortcodes for you to use in your theme. The plugin is not a requirement to use with Ruven Themes, but it will extend the themes to function as you see them in the demos.

= Can I use this plugin with other themes? =

Ruven Toolkit was developed to extend the functionality of Ruven Themes, however you could use it with other themes. It creates Portfolio and Gallery post types, adds custom widgets, a post format UI, and shortcodes, which are fairly universal in the theme world. You may have to provide some extra styling to customize it to your needs.

==Installation==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Ruven Toolkit" into the Search input and click the "Search Plugins" button.
3. Locate the Ruven Toolkit in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. Gallery and Portfolio Items will show up in the admin menu, Widgets will be available in Appearance &rarr; Widgets, and Shortcodes will be available in the post editor toolbar by clicking the flash icon.

It can also be installed manually.

1. Download the "Ruven Toolkit" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Ruven Toolkit" in the list and click the "Activate" link.
5. Portfolio Items will show up in the admin menu, Widgets will be available in Appearance &rarr; Widgets, and Shortcodes will be available in the post editor toolbar by clicking the flash icon.
